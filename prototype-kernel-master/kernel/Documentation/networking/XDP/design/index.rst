Design
======

.. (comment) write something about performance and programmability
   Took below from Tom's presentation on XDP

XDP is designed for high performance. It uses known techniques and
applies selective constraints to achieve performance goals.  XDP is
also designed for programmability. New functionality can be
implemented on the fly without needing kernel modification
